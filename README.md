# Generadores pseudoaleatorios

Aplicación de escritorio en Python para generar 1000 valores pseudoaleatorios con métodos congruenciales y no congruenciales.

## Métodos incluidos

- Congruencia Lineal Mixto
- Congruencial Multiplicativo
- Congruencial Aditivo
- Cuadrados Medios
- Producto Medio
- Fibonacci

## Instalación

```bash
pip install PySide6 matplotlib
```

## Ejecución

```bash
python main.py
```

La aplicación usa PySide6 para la interfaz y matplotlib para el histograma integrado. No usa Tkinter.
