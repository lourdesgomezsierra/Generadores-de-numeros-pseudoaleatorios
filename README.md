# Generadores pseudoaleatorios

Aplicación de escritorio en Python para generar 1000 valores pseudoaleatorios con métodos congruenciales y no congruenciales.

## Métodos incluidos

- Congruencia Lineal Mixto
- Congruencial Multiplicativo
- Congruencial Aditivo
- Cuadrados Medios
- Producto Medio
- Fibonacci

## Instalación

```bash
pip install PySide6 matplotlib numpy scipy
```

## Ejecución

```bash
python main.py
```

La aplicación usa PySide6 para la interfaz, matplotlib para gráficos integrados y SciPy para los valores críticos de las pruebas estadísticas. No usa Tkinter.
